How to Join

  1. Install Prism Launcher
  2. Create a new instance
  3. Select CurseForge, search Craftoria, and select OK
  4. Click Open Missing to open ~12 mods in chrome. Switch to chrome and load each tab so the files download.
  5. Make sure all the mod links are green then hit OK.
  6. Double click the new Craftoria instance to launch!
